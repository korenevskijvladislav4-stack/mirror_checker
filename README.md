# Mirror Update (Node.js + Express + Google Sheets)

Сервис читает список зеркал из Google Sheets, раз в сутки (в `00:00` по МСК) проверяет доступность `Зеркало 2` и `Зеркало 3` и при необходимости запускает обновление, заменяя домен на следующий номер.

## Таблица Google Sheets

Ожидается заголовок (первая строка) с колонками:

`Проект | Зеркало 1 | Зеркало 2 | Зеркало 3`

`Зеркало 1` считается постоянным источником.

## Нумерация зеркал

URL имеют вид:

- `https://roxcasino1567.com`
- `https://fresh357.com`

Если `Зеркало 2` или `Зеркало 3` недоступно, берём максимум номера из текущих `Зеркало 2/Зеркало 3` и заменяем упавшее зеркало на значение `max + 1` (с тем же префиксом/суффиксом, если URL парсится).

## Запуск

1. Создай service account в Google Cloud, выдай доступ к Google Sheet.
2. Скопируй `.env.example` в `.env` и заполни:
   - `GOOGLE_SERVICE_ACCOUNT_KEYFILE`
   - `GOOGLE_SPREADSHEET_ID`
   - `GOOGLE_SHEET_NAME`
   - `MIRROR_REFRESH_COMMAND_TEMPLATE` (как обновлять таргет из сорса)
3. Установи зависимости и запусти сервис:

```bash
npm i
npm run dev
```

## API

- `GET /health`
- `POST /run-once` (ручной запуск обновления)

## Важно

Команда обновления зеркал (`MIRROR_REFRESH_COMMAND_TEMPLATE`) должна быть задана так, чтобы сервис мог вызвать обновление таргета из источника.
По умолчанию в коде “обновление” не реализовано, пока ты не укажешь шаблон команды.
